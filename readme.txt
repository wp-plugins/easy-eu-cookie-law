The easiest way to comply with the new European cookie law.

== Description ==

Cookie law is a Wordpress Plugin that helps you to comply with the new European cookie law by the easiest way.

This plugin has different alert colors and has been translated in English and Spanish.

Not database is required.

== Installation ==

1. Upload `cookie_law` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Configure your settings in the 'Cookie law' menu.
